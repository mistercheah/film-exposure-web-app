Fixes in this version
- Fixed a JavaScript error that broke:
  - image selection / analysis
  - switching between Input and Result tabs
- Keeps the v9 workflow:
  - Save Result
  - Save Settings
  - Export History (JSON + CSV)

How to update on GitHub / Vercel
1. Unzip this package.
2. Open your GitHub repository.
3. Upload and replace:
   - index.html
   - styles.css
   - script.js
   - README.txt
4. Commit the changes.
5. Vercel will redeploy automatically.
