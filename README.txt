Film Exposure Assistant - Deployable Web Prototype

What is included
- index.html
- A single-file prototype web app you can upload directly to Vercel, Netlify, or any static host.

What it does
- Input screen for film, ISO, filter, lighting, aperture, compensation, metering mode, and image upload
- Result screen for metered exposure, reciprocity status, and final exposure
- Placeholder calculation logic for Phase 1

Important note
- The uploaded image is part of the UI flow only in this package.
- Automatic image analysis is not implemented yet.
- Reciprocity values are simplified placeholders for prototyping.

Fastest way to host on Vercel
1. Create a Vercel account.
2. Create a new project.
3. Drag and drop the whole folder, or upload index.html.
4. Deploy.
5. Open the generated URL on your Android phone.
6. In Chrome on Android, use 'Add to Home Screen' if you want it to behave like an app.

Alternative
- You can also double-click index.html to open it locally in your browser.
