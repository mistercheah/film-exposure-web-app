Film Exposure Web App v9.4

Fixes in this version
- Live Summary now shows Practical Exposure instead of the raw calculated final exposure
- Live Summary practical exposure now matches the Result screen
- Time formatting remains handled by the same seconds-to-label formatter, including minutes/seconds for long exposures
- Current version shown in the app: v9.4

How to update on GitHub / Vercel
1. Unzip this package.
2. Open your GitHub repository.
3. Upload and replace:
   - index.html
   - styles.css
   - script.js
   - README.txt
4. Commit the changes.
5. Vercel will redeploy automatically.
