Film Exposure Web App v2

What is new in this version
- Added simple image brightness analysis from uploaded photo
- Added Auto from uploaded image as default lighting source
- Shows estimated scene EV in the input summary and result screen
- Keeps manual lighting presets as fallback
- Keeps filter and reciprocity logic

How to update on GitHub / Vercel
1. Unzip this package.
2. Open your GitHub repository.
3. Upload and replace:
   - index.html
   - styles.css
   - script.js
4. Commit the changes.
5. Vercel will redeploy automatically.

Local testing
- Open index.html in your browser.
